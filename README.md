# Portfolio - Gael Le Reun

Bienvenue sur le dépôt de mon portfolio professionnel. Ce site est conçu pour présenter mes projets, mes compétences techniques en IA & Data, ainsi que mon parcours d'étudiant ingénieur.

### [Accéder au site](https://gael926.github.io/portfolio-gael-le-reun/)

---

## Points forts du projet

- **Design System v2** : Une interface moderne, épurée et texturée, privilégiant la lisibilité et l'impact visuel.
- **Thème Sombre & Clair** : Support complet du mode sombre avec détection automatique des préférences système.
- **Bilingue (FR/EN)** : Système de traduction robuste intégré permettant de basculer instantanément entre le français et l'anglais.
- **100% Responsive** : Expérience utilisateur optimisée pour tous les supports (Mobile, Tablette, Desktop).
- **Performances** : Code Vanilla ultra-léger (pas de framework lourd), optimisation SEO et accessibilité soignée.

## Technologies & Outils

- **Core** : HTML5, CSS3 modulaire, JavaScript (ES6+)
- **Styling** : [Tailwind CSS](https://tailwindcss.com/) pour la rapidité et la cohérence.
- **Iconographie** : [Lucide Icons](https://lucide.dev/) pour un rendu net et moderne.
- **Typographie** : Sora (Display) & IBM Plex Mono (Labels).
- **Déploiement** : GitHub Pages avec intégration continue.
